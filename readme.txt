== Description ==

This plugin adds the required javascript for paletly widget. After registering at http://bloggers.paletly.com/, bloggers can obtain their IDs (Blog Id, Sub Id) for plugin settings configuration. After activating plugin you can see paletly shop similar widget below your blog images within 48 hours. This plugin enables fashion bloggers to activate the widget by simply putting in their unique blog ID and sub ID.

Paletly's 'Shop Similar' widget is the best way to monetize fashion images on your blog.

The non-intrusive shopping carousel features similar products from over 600 global brands.

Use this simple 4-step process to provide your readers fulfillment at the point of inspiration.

* **SIGN UP**: Register and we will provide you a single snippet of code to paste in your blog's html code.

* **ACTIVATE**:Sit back while we active shopping windows for relevant images on your blog.

* **SELL**:Your readers are now connected to over 600 brands & a million products.

* **EARN**:On every purchase, you earn up to 10% in sales commissions.

For more information visit:

[Paletly bloggers](http://www.paletly.com/bloggers)

== Installation ==

1. Upload `paletly` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin in the WordPress Settings > Paletly menu (Admin > Settings > Paletly)
4. To fill `blog id` and `sub id` you need to register with paletly. You can join us at [bloggers.paletly.com](http://bloggers.paletly.com)
